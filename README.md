# RL-SSL
Code and data for "RL-SSL: Robust Encrypted Traffic Classification with Limited Labeled Data via Semi-Supervised Contrastive Learning"

**Dataset**
1. CIC-IoT2023: https://www.unb.ca/cic/datasets/iotdataset-2023.html
2. CipherSpectrum: https://cgi.cse.unsw.edu.au/~cspectrum/
3. USTC-TFC2016: https://github.com/davidyslu/USTC-TFC2016
4. ISCX-VPN2016: https://www.unb.ca/cic/datasets/vpn.html
5. ISCX-Tor2016: https://www.unb.ca//cic/datasets/tor.html

The data processed by the Traffic Embedding and Augmentation (TEA) module is stored in Data/features.npz.
